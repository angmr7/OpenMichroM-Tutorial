## Running locally

### Required packages

For this example i am using OpenMichroM 1.0.7

```
conda install -c conda-forge OpenMiChroM==1.0.7
```

Or with pip

```
pip install OpenMiChroM==1.0.7
```

And matplotlib and numpy

```
pip install matplotlib
pip install numpy
```
